import { Check, Rocket, Settings, Code } from 'lucide-react';

The `fumadocs-enhanced-steps` package provides an enhanced Steps component with support for different counter types, custom icons, and step titles.

## Basic Usage

<Steps>
  <Step title="Install the package">
    Run the following command to install:
    ```bash
    npm install fumadocs-enhanced-steps
    ```
  </Step>
  <Step title="Import the CSS">
    Add the CSS import to your global styles:
    ```css
    @import 'fumadocs-enhanced-steps/styles/steps.css';
    ```
  </Step>
  <Step title="Use in your MDX">
    Add the components to your MDX configuration and start using them!
  </Step>
</Steps>

## Lettered Steps (lowercase)

Use `type="lower-alpha"` for lowercase letters:

<Steps type="lower-alpha">
  <Step title="First item">
    This is step a.
  </Step>
  <Step title="Second item">
    This is step b.
  </Step>
  <Step title="Third item">
    This is step c.
  </Step>
</Steps>

## Lettered Steps (uppercase)

Use `type="upper-alpha"` for uppercase letters:

<Steps type="upper-alpha">
  <Step title="Section A">
    Introduction and overview.
  </Step>
  <Step title="Section B">
    Main content and examples.
  </Step>
  <Step title="Section C">
    Summary and next steps.
  </Step>
</Steps>

## Roman Numerals

Use `type="lower-roman"` or `type="upper-roman"`:

<Steps type="upper-roman">
  <Step title="Chapter I">
    The beginning of our journey.
  </Step>
  <Step title="Chapter II">
    The challenges we face.
  </Step>
  <Step title="Chapter III">
    The triumphant conclusion.
  </Step>
</Steps>

## Custom Icons

Replace numbers with custom icons using the `icon` prop:

<Steps>
  <Step title="Configure" icon={<Settings className="size-3.5" />}>
    Set up your project configuration.
  </Step>
  <Step title="Develop" icon={<Code className="size-3.5" />}>
    Write your code and components.
  </Step>
  <Step title="Deploy" icon={<Rocket className="size-3.5" />}>
    Ship it to production!
  </Step>
  <Step title="Done" icon={<Check className="size-3.5" />}>
    Celebrate your success!
  </Step>
</Steps>

## Without Titles

Steps work without titles too:

<Steps>
  <Step>
    First, clone the repository.
  </Step>
  <Step>
    Then, install the dependencies.
  </Step>
  <Step>
    Finally, start the development server.
  </Step>
</Steps>

## Counter Types Reference

| Type | Output |
|------|--------|
| `decimal` (default) | 1, 2, 3... |
| `lower-alpha` | a, b, c... |
| `upper-alpha` | A, B, C... |
| `lower-roman` | i, ii, iii... |
| `upper-roman` | I, II, III... |
